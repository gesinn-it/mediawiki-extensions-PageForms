= Quick Start Guide
:toc: left
include::../attributes.adoc[]

This guide walks through setting up a Page Forms-based wiki from scratch.
The starting point is a running MediaWiki installation with Page Forms (and optionally Semantic MediaWiki) installed.

The first step is to *figure out your data structure*: what types of pages will the site have, and what data does each page type hold?
Once that is clear, the necessary wiki pages can be created — properties (if using SMW), templates, forms, and categories.

== Step-by-step setup

=== 1. Create properties (SMW only)

If Semantic MediaWiki is installed, create a page in the `Property:` namespace for each piece of data you want to store semantically.
Each property page defines the property's type (Page, Text, Number, Date, etc.) and optionally its allowed values.

Example: create the page `Property:Was written by` with type "Page".

=== 2. Create templates

A template controls how a page type is displayed and how its data is stored.
Create a template page in the `Template:` namespace.
The template receives field values as parameters and renders them as wikitext.

When used with SMW, the template also sets property values using `#set` or `#subobject`.

Example: create `Template:Book` with parameters for author, genre, year, and number of pages.

=== 3. Create forms

A form definition page lives in the `Form:` namespace and specifies which templates and fields appear in the editing form.
See xref:../admin/defining-forms.adoc[Defining Forms] for the complete syntax.

Example: create `Form:Book` that maps to `Template:Book`.

=== 4. Create categories

Create a category page for each page type and associate it with the appropriate form using `#default_form`:

[source]
----
{{#default_form:Book}}
----

This makes every page in the category editable via the "Edit with form" tab.
See xref:../admin/linking-to-forms.adoc[Linking to Forms] for other ways to associate forms with pages.

=== 5. Enable links to forms

Add links in the sidebar and on relevant pages so that users can reach the forms to create new pages.
Use `#formredlink` inside templates to have red links on pages point directly to the right creation form.

=== 6. Add data

Navigate to `Special:FormEdit/FormName` or use the form links created in step 5 to start entering pages through forms.

== Example: a book and author wiki

The following example builds a small wiki for tracking books and authors.

=== Data structure

Two page types:

* *Book* — title (page name), author(s), genre(s), year of publication, number of pages
* *Author* — name (page name), country of origin, list of books

=== Properties

Create the following property pages:

* `Property:Was written by` — type: Page (list)
* `Property:Has genre` — type: Text (list), allowed values: Art, Cookbook, Fiction, History, Poetry, Science
* `Property:Was published in year` — type: Number
* `Property:Has number of pages` — type: Number
* `Property:Is from country` — type: Text

=== Template:Book

[source]
----
<noinclude>
{{#template_params:
Authors|Genres|Year|Number of pages
}}
This template stores book data.
[[Category:Books]]
</noinclude><includeonly>
{{#set:Was written by={{{Authors|}}}|+sep=,}}
{{#set:Has genre={{{Genres|}}}|+sep=,}}
{{#set:Was published in year={{{Year|}}}}}
{{#set:Has number of pages={{{Number of pages|}}}}}
; Author(s): {{#arraymap:{{{Authors|}}}|,|x|{{#formredlink:target=x|form=Author}}{{#set:Was written by=x}}}}
; Genre(s): {{{Genres|}}}
; Year: {{{Year|}}}
; Pages: {{{Number of pages|}}}
[[Category:Books]]
</includeonly>
----

=== Template:Author

[source]
----
<noinclude>
{{#template_params:
Country
}}
This template stores author data.
</noinclude><includeonly>
{{#set:Is from country={{{Country|}}}}}
; Country: {{{Country|}}}
=== Books by this author ===
{{#ask:[[Was written by::{{PAGENAME}}]]|format=ul}}
[[Category:Authors]]
</includeonly>
----

=== Form:Book

[source]
----
{{{info|edit title=Edit book}}}
{{{for template|Book}}}
{| class="formtable"
! Author(s):
| {{{field|Authors|list|input type=tokens|values from category=Authors}}}
|-
! Genre(s):
| {{{field|Genres|list|input type=checkboxes}}}
|-
! Year of publication:
| {{{field|Year}}}
|-
! Number of pages:
| {{{field|Number of pages}}}
|}
{{{end template}}}
{{{standard input|save}}} {{{standard input|cancel}}}
----

=== Form:Author

[source]
----
{{{info|edit title=Edit author}}}
{{{for template|Author}}}
{| class="formtable"
! Country of origin:
| {{{field|Country}}}
|}
{{{end template}}}
{{{standard input|save}}} {{{standard input|cancel}}}
----

=== Category:Books and Category:Authors

On the `Category:Books` page:

[source]
----
{{#default_form:Book}}
----

On the `Category:Authors` page:

[source]
----
{{#default_form:Author}}
----

=== Sidebar links

Add entries to `MediaWiki:Sidebar` to link to the forms for creating new books and authors, and to the category pages for browsing existing ones.
